Для корректной работы скрипта требуется налицие папки car-logs в текущей дериктории
Все файлы внутри кар логс должны иметь расширение .csv
Скрипт на выходе создает папку results в которой лежит файл total.log с результатами для всех 
файлов и лежат *.log файлы для каждого .csv с посчитанными значениями
